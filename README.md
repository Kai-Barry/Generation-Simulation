# Algorithm Sandbox
Both teams have the specific tiles that perform different algorithms to multiply. These algorithms fight against each other in a sandbox enviroment.

Left Click: Place Tile

Right Click: Remove Tile

F: Cycle Tile

E: Singular tick

R: Continuous ticks
